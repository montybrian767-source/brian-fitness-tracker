# Brian Fitness Tracker v15.2 — True Mockup Layout

This build is a layout rebuild designed to look closer to the investor-style mockup.

## New in v15.2

- Dark navy sidebar
- Full-width workout screen
- White exercise cards
- Exercise visual panels
- No expander boxes on the workout page
- Table-style set rows
- Right-side workout summary
- Muscle group panel
- Quick actions and tips
- Completed workouts save to `data/workout_log.csv`
- `data/workouts.csv` remains the workout plan/template

## Run locally

Double-click `run_app.bat`.

## Streamlit Cloud

Upload all files/folders to GitHub, then reboot/redeploy your Streamlit app.

Important: do not delete `data/workout_log.csv` if you already have saved workouts.
