# Brian Fitness Tracker Pro v13 — Adaptive Training Engine

This version adds 4-week workout rotations for LA Fitness.

## Features
- Same weekly muscle groups
- Exercises rotate every 4 weeks
- Block 1: Weeks 1-4
- Block 2: Weeks 5-8
- LA Fitness exercise library
- Knee-safe leg rehab rules
- Workout builder page
- Progress charts
- Workout history CSV

## Run on Windows
1. Unzip this folder.
2. Double-click `run_app.bat`.
3. The app opens in your browser.

## Use at the gym
Deploy to Streamlit Cloud to open it anywhere on your phone.
