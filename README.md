# Brian Fitness Tracker Pro v20 — Commercial Release Foundation

This build keeps the v19 Reports/Coach/Nutrition/Recovery system and adds a **Commercial Launch Center**.

## New in v20
- Commercial Launch Center
- Investor-ready product checklist
- Release roadmap inside the app
- Pricing/subscription planning section
- Keeps Reports Center
- Keeps Coach Mode
- Keeps Nutrition + Recovery
- Keeps AI Workout Rotation Engine

## Run locally
Double-click `run_app.bat` or run:

```bash
python -m streamlit run app.py
```

## Streamlit Cloud
Upload the full folder contents to GitHub and redeploy your Streamlit app.

## Important
This is the commercial foundation. For real paying users, the next critical upgrade is a real cloud database/login system, such as Supabase.
